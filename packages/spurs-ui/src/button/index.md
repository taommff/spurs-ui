## 属性

| 属性     | 说明             | 类型                            | 默认值  |
| -------- | ---------------- | ------------------------------- | ------- |
| type     | 设置按钮类型     | `default` \|`primary` \|` dashed` | default |
| size     | 设置按钮大小     | `default` \| `samll` \| `large`   | default |
| disabled | 设置按钮禁用状态 | Boolean                         | false   |

## 事件

| 事件  | 说明               | 类型                      |
| ----- | ------------------ | ------------------------- |
| click | 点击时候触发该事件 | `(event： Event) => void` |
